Expense Tracker

A modern and responsive Expense Tracker web application built using React.js and Tailwind CSS. This project helps users efficiently manage their daily expenses by adding, filtering, and tracking transactions in a clean UI.

Features
Add new expenses with title, amount, and category
Delete expenses
Filter expenses (e.g., All, Income, Expense)
Data persistence using Local Storage
Fast and responsive UI
Fully responsive design

Tech Stack
Frontend: React.js
Styling: Tailwind CSS
State Management: React Hooks / Context API (if used)
Storage: Browser Local Storage
